# YouTube Playlist Audio Downloader (MP3) 🎶

Современное веб-приложение для скачивания аудиодорожек из плейлистов YouTube с автоматической фильтрацией, редактированием метаданных и сохранением истории.

## 🌟 Возможности

1. **Красивый и отзывчивый интерфейс**:
   - Премиальный дизайн в темных тонах с размытием (glassmorphism), плавными градиентами и микро-анимациями.
   - Разделы «Загрузчик» и «История» загрузок.
   
2. **Умная фильтрация треков**:
   - **Замедленные версии (Slowed)**: Автоматически обнаруживает замедленные треки (по ключевым словам `slowed`, `reverb` и др.), снимает с них галочки и выводит кнопку **«Найти оригинал»**.
   - **Поиск оригиналов**: Встроенный поиск альтернативных (оригинальных) версий на YouTube с возможностью мгновенной замены прямо в списке плейлиста.
   - **Длинные видео**: Автоматически снимает галочки с видео продолжительностью более 10 минут, предупреждая пользователя.

3. **Редактирование метаданных на лету**:
   - Интеллектуальное разделение названия видео на Исполнителя и Трек.
   - Возможность вручную скорректировать Исполнителя и Название трека прямо в карточке перед началом скачивания.
   
4. **Загрузка и конвертация**:
   - Загрузка с помощью `yt-dlp` и конвертация в формат MP3 (320 kbps) с помощью встроенного или системного `FFmpeg`.
   - Полностью асинхронный процесс: отображение прогресса скачивания, скорости и ETA для каждого файла в реальном времени.

5. **История скачиваний**:
   - Сохранение истории в локальную базу данных SQLite (`~/.yt_downloader_history.db`).
   - Отображение даты скачивания, ссылок и путей сохранения с удобной кнопкой быстрого копирования пути.

6. **Автономность и Автоматический FFmpeg**:
   - Приложение автоматически проверяет наличие `ffmpeg.exe` и `ffprobe.exe` в папке `bin/` или в системе.
   - Если FFmpeg отсутствует, его можно скачать в один клик прямо из веб-интерфейса!

## 🛠️ Установка и запуск

### Шаг 1. Установите зависимости
Убедитесь, что у вас установлен Python (версии 3.7+). Установите необходимые библиотеки:
```bash
pip install -r requirements.txt
```

### Шаг 2. Запуск приложения
Запустите основной скрипт:
```bash
python main.py
```
После запуска сервер автоматически откроет страницу приложения в вашем браузере по адресу: [http://127.0.0.1:8088](http://127.0.0.1:8088).

---

## 📦 Сборка в один EXE-файл (Windows)

Для компиляции приложения в переносимый исполняемый файл `.exe` запустите файл:
```bash
build.bat
```
Скрипт автоматически установит `PyInstaller` и соберет проект. Готовый файл появится в папке `dist/main.exe`. Он будет содержать в себе все статические файлы веб-интерфейса и исполняемые файлы FFmpeg, поэтому его можно запускать на любом компьютере без установки Python!

* [main.py](file:///D:/.code/youtube downloader/main.py) — Точка входа в приложение, запуск сервера и автоматическое открытие браузера.
* [app/](file:///D:/.code/youtube downloader/app/) — Логика бэкенда на Python:
  * [api.py](file:///D:/.code/youtube downloader/app/api.py) — Эндпоинты FastAPI для взаимодействия фронтенда с системой.
  * [downloader.py](file:///D:/.code/youtube downloader/app/downloader.py) — Взаимодействие с `yt-dlp` и `FFmpeg` (анализ, поиск оригиналов, скачивание).
  * [history.py](file:///D:/.code/youtube downloader/app/history.py) — Хранение истории загрузок в SQLite.
* [static/](file:///D:/.code/youtube downloader/static/) — Веб-интерфейс (HTML5, CSS3, JavaScript):
  * [index.html](file:///D:/.code/youtube downloader/static/index.html) — Разметка интерфейса.
  * [style.css](file:///D:/.code/youtube downloader/static/style.css) — Стили, темы и адаптивный дизайн.
  * [app.js](file:///D:/.code/youtube downloader/static/app.js) — Интерактивная логика на клиенте.
* [bin/](file:///D:/.code/youtube downloader/bin/) — Папка для размещения `ffmpeg.exe` и `ffprobe.exe`.
